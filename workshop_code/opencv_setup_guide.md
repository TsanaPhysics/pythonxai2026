# 📷 คู่มือเตรียม OpenCV: สำหรับนักนวัตกรรม
**Agri-Vision Setup Guide (2026)**

OpenCV เป็นหัวใจสำคัญของการทำให้ AI "มองเห็น" และแยกแยะรอยโรคบนใบพืชได้

---

## 🛠️ การติดตั้งที่แนะนำ

### 1. สำหรับ JupyterLab (รันได้ทันที)
JupyterLab มี OpenCV ติดตั้งมาให้แล้ว แต่มีข้อจำกัดเรื่องการแสดงผลภาพแบบหน้าต่าง (`cv2.imshow()`) ให้แก้ไขโดยใช้:
```python
from google.jupyter.patches import cv2_imshow
cv2_imshow(image)
```
*หรือใช้ `vision_helper.py` ที่เตรียมไว้ให้*

### 2. สำหรับเครื่องคอมพิวเตอร์ส่วนตัว (Local)
ติดตั้งผ่าน Terminal หรือ CMD:
```bash
pip install opencv-python opencv-contrib-python
```

---

## 🚨 จุดที่ต้องระวัง (Troubleshooting)

| ปัญหาที่พบ | วิธีการแก้ไข |
| :--- | :--- |
| **เปิดกล้องไม่ได้** | ตรวจสอบ Permission ของกล้องที่ระดับ OS (macOS/Windows) |
| **หาไฟล์ภาพไม่เจอ** | เช็ค Path ของไฟล์ `images/plant.png` ให้ถูกต้อง |
| **HSV ไม่แม่นยำ** | แสงในห้องแล็บมีผลต่อค่า Hue ต้องปรับแต่งขีดจำกัด (Low/High) ใหม่ |

---
> [!TIP]
> **พี่ยักษ์บอกต่อ**: เพื่อความรวดเร็วในการสอน แนะนำให้รันโค้ดแสดงผลผ่าน `vision_helper.py` เพื่อให้แน่ใจว่าได้มาตรฐานเดียวกันทั้งห้องแล็บครับ!
