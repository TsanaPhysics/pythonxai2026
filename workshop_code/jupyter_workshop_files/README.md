# 📂 คลังโค้ดตัวอย่าง: Python x AI 2026
ยินดีต้อนรับสู่แหล่งรวมโค้ดปฏิบัติการจริงสำหรับการอบรม 12 ชั่วโมง โดย ผศ.ดร.ชีวะ ทัศนา

---

## 🏗️ โครงสร้างโฟลเดอร์
- `python_scripts/`: ไฟล์นามสกุล `.py` สำหรับรันบนเครื่องคอมพิวเตอร์ผ่าน Terminal หรือ IDE (VS Code, PyCharm)
- `notebooks/`: ไฟล์นามสกุล `.ipynb` สำหรับเปิดบน JupyterLab หรือ Jupyter Notebook
- `datasets/`: ไฟล์ข้อมูลตัวอย่าง (CSV) สำหรับการทดลอง Machine Learning

---

## 🚀 วิธีการรันโค้ด (How to Run)

### 1. การรันไฟล์ Python (.py)
- เปิด Terminal หรือ CMD
- พิมพ์ `python ชื่อไฟล์.py` (เช่น `python ex01_check_system.py`)
- **พิเศษ**: สำหรับบทที่ 6 (Pygame Zero) ให้รันด้วย `pgzrun ชื่อไฟล์.py`

### 2. การรันบน JupyterLab (.ipynb)
- เข้าไปที่ [jupyter.org/try-jupyter/lab/](https://jupyter.org/try-jupyter/lab/)
- เลือกเมนู **Upload** แล้วเลือกไฟล์จากโฟลเดอร์ `notebooks/`
- กดปุ่ม **Play** ในแต่ละเซลล์เพื่อดูผลลัพธ์
